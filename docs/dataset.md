# Dataset Specification

## Overview
The dataset for this project is entirely synthetic, representing realistic automotive properties with 14 distinct input features. It is designed to evaluate nonlinear regression capabilities by incorporating complex interactions and physics-inspired target formulas.

## Features

| # | Feature | Type | Range | Distribution | Physical Meaning |
|---|---------|------|-------|--------------|------------------|
| 1 | `engine_size` | float | 0.8–6.5 L | Lognormal(μ=0.7, σ=0.35) | Engine displacement in litres |
| 2 | `horsepower` | int | 80–700 HP | Gamma(shape=4, scale=45)+70 | Engine power output |
| 3 | `weight` | int | 900–3000 kg | Normal(μ=1500, σ=350) | Vehicle curb weight |
| 4 | `cylinders` | int | {3,4,5,6,8,10,12} | Categorical: probs [.05,.40,.03,.28,.18,.02,.04] | Number of engine cylinders |
| 5 | `age` | int | 0–30 years | Exponential(scale=6) | Vehicle age |
| 6 | `turbo_pressure` | float | 0 or 0.3–3.0 bar | 45% chance turbocharged, Normal(μ=1.2, σ=0.4) | Turbocharger boost pressure |
| 7 | `fuel_octane` | int | {87,89,91,93,95} | Categorical: probs [.35,.25,.20,.12,.08] | Fuel octane rating |
| 8 | `drivetrain_ratio`| float | 2.0–5.5 | Normal(μ=3.5, σ=0.6) | Final drive gear ratio |
| 9 | `compression_ratio`| float | 8.0–14.0 | Normal(μ=10.5, σ=1.2) | Engine compression ratio |
| 10| `aerodynamic_drag`| float | 0.22–0.50 | Normal(μ=0.32, σ=0.05) | Drag coefficient (Cd) |
| 11| `tire_width` | int | 175–315 mm | Normal(μ=225, σ=25) | Tire section width |
| 12| `ambient_temp` | float | -10–45 °C | Normal(μ=20, σ=10) | Ambient temperature |
| 13| `altitude` | int | 0–3000 m | Exponential(scale=400) | Operating altitude |
| 14| `ethanol_blend` | int | {0,10,15,85} | Categorical: probs [.40,.35,.15,.10] | Ethanol percentage in fuel |

## Target Variable: MPG (Miles Per Gallon)
The target is generated by a complex, physics-inspired formula with approximately 35 terms, making it a perfect candidate for testing nonlinear regression models. The formula incorporates:

1. **Core Physics**: Otto cycle thermal efficiency approximations.
2. **Nonlinearities**: Exponential, logarithmic, and trigonometric transformations applied to features.
3. **Cross-Feature Interactions**: 
   - `turbo_pressure` × `engine_size`
   - `fuel_octane` × `horsepower`
   - `weight` × `drivetrain_ratio`
4. **Environmental Effects**:
   - `altitude` effects on air density
   - `ambient_temp` effects on thermal efficiency and warmup cycles
5. **Noise Model**: 
   - A base Gaussian noise ($\sigma=2.5$).
   - A heteroscedastic component (noise variance increases for older and heavier vehicles).

This extreme nonlinearity means classical linear models will significantly underperform without extensive manual feature engineering.
